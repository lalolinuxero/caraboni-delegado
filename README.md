# 🚀 Candidatura Delegado Estudiantil: Eduardo Caraboni

¡Bienvenido al repositorio oficial de mi campaña para **Delegado de Ingeniería en Ciberseguridad** (Primer Año)! 🛡️

Este sitio web fue diseñado con una estética **Retro-Windows XP** para combinar la nostalgia tecnológica con la seriedad y transparencia que nuestra carrera requiere.

🌐 Ver en vivo
Puedes visitar la página oficial aquí:
👉 **[https://delegado.eduardocaraboni.space](https://delegado.eduardocaraboni.space)**

## 📂 Estructura del Proyecto
* `/index.html`: Puerta de entrada principal.
* `/img/`: Assets gráficos (incluyendo el icónico wallpaper de Windows XP y elementos de INACAP).
* `/fonts/`: Tipografías personalizadas para mantener la identidad visual.
  
🎯 Objetivos de la Campaña
1.  **Comunicación Directa:** Uso de tecnologías abiertas para mantener a todos informados.
2.  **Transparencia:** Código fuente abierto para que cualquiera pueda auditar la web.
3.  **Escucha Activa:** Implementación de muros virtuales para propuestas de los compañeros.
